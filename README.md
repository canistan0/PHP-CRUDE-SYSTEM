# PHP-CRUDE-SYSTEM
A simple PHP CRUD application for managing users using PDO, featuring a responsive table UI, delete confirmation, and status alerts for operations.

## Features
 
- Display users in a responsive table
- Add new users
- Edit existing users
- Delete users with confirmation prompt
- PDO-based database interaction (secure & reusable)
- Alert messages for success/error states

  ---

## Technologies Used
 
- PHP (Core PHP with PDO)
- MySQL
- HTML5 / Bootstrap
- JavaScript (jQuery for confirmation)
